Challenge Overview

Welcome to Bamboozled, a unique coding challenge event at TECHTONIC 2K25 where your typing skills and problem-solving abilities will be put to the ultimate test!

The Twist

In this challenge, the ASCII values of keyboard keys are formulated and shuffled. You won't know which key is mapped to which value, but you'll be provided with hints about the formula used for shuffling. Can you decode the pattern and complete the tasks?

Competition Format

This one-day competition features progressively difficult rounds where participants must type code using a special-purpose keyboard with shuffled keys:

ROUND 1: The Warmup

Type one specified line of Python/C code in just 10 minutes using the shuffled keyboard. Those who complete the task correctly will advance to the next round.

ROUND 2: Ultimate Test

The difficulty increases as you must type a small code snippet without any spelling or syntax errors within 15 minutes. Successful participants will proceed to the final round.
